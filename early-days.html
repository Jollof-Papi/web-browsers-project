Birth of the Browser

In 1990, Tim Berners-Lee created the first browser, called Worldwide Web (later Nexus), to navigate the young World Wide Web. In 1993, Mosaic introduced graphics alongside text, making the web visually friendly.

The First Browser War

In 1995, Microsoft released Internet Explorer, bundled with Windows, starting the "browser wars" with Netscape Navigator. By 2001, Internet Explorer had over 90% market share, leading to antitrust investigations. This era saw innovation like JavaScript from Netscape and ActiveX from Microsoft, despite compatibility and security challenges.
